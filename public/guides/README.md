# Guide Assets Convention

Each guide stores assets at: `/public/guides/{guideId}/`

## Required files by tier:

### Tier 1 (Pillar Guides)
- `orientation.jpg` — Calm grounding image after intro
- `checklist.jpg` — Visual checklist/comparison at decision point  
- `overview.pdf` — Downloadable full guide PDF
- `infographic.png` — Visual summary infographic

### Tier 2 (Supporting Guides)
- `orientation.jpg` — Grounding image
- `infographic.png` — Required visual summary

### Tier 3 (Micro/Stories)
- `orientation.jpg` — Emotional tone image
- `clarity.jpg` — Resolution image

## Naming rules:
- All lowercase, no spaces
- `guideId` matches the registry ID exactly (snake-case with hyphens)
- Images: `.jpg` for photos, `.png` for infographics/checklists
- Documents: `.pdf` only
